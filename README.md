# Vacation Tracker
A simple vacation request tracker built with Node/Express and MongoDB.
## Features
* Request submission
* Approval workflow
* User dashboard
* Admin dashboard
## Installation
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with your MongoDB URI and port
4. Run `npm start`
## API Endpoints
* `GET /api/requests`: Fetch all requests
* `POST /api/requests`: Create a new request
* `GET /api/users`: Fetch all users
